This is a web api for getting custom fractal images with a web request

You can check out the source code here

Currently contains most of the features I planned to have including:

  • Customizable colors
  • Customizable position, zoom
  • Customizable image dimensions
Currently contains the following fractals:
  • Mandelbrot set (z^2 + c)
  • Burning ship (abs(z)^2 + c)
  • Tricorn (conj(z)^2 + c)
  • Feather ((z^3 / 1 + z * z) + c)
  • Eye ((z/c)^2 - c)

To get an image you can simply do a get request on https://fractal.ciubix.xyz/fractals/"Name of the fractal here".
Here's some examples
  • fractals.ciubix.xyz/fractals/Mandelbrot:
  • You can also specify things like image dimensions
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256:
  • Position
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256&position_x=-1.5&position_y=-.05:
  • Zoom
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256&position_x=-1.5&position_y=-.05&zoom=10:
  • Maximum number of iterations (for more information check out this article on wikipedia)
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256&position_x=-1.5&position_y=-.05&zoom=10&max_iterations=2000:
  • Colors of the fractal, specified in form of hex without the #
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256&position_x=-1.5&position_y=-.05&zoom=10&max_iterations=2000&colors=e50000,ff8d00,ffee00,028121,004cff,770088:
  • Number of colors used by the coloring algorithm (explained here)
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256&position_x=-1.5&position_y=-.05&zoom=10&max_iterations=2000&colors=e50000,ff8d00,ffee00,028121,004cff,770088&num_colors=150:
  • Anti aliasing, up to 255, but PLEASE keep this low unless absolutely necessary, my server may actually crash if that number is too high
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256&position_x=-1.5&position_y=-.05&zoom=10&max_iterations=2000&colors=e50000,ff8d00,ffee00,028121,004cff,770088&num_colors=150&msaa=10:
  • Turn on smooth coloring algorithm (for more detail see this great article by Inigo Quilez)
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256&position_x=-1.5&position_y=-.05&zoom=10&max_iterations=2000&colors=e50000,ff8d00,ffee00,028121,004cff,770088&num_colors=150&msaa=10&smooth=true:
  • In case you're trying to find the exact position where you want to be you can turn on debug, which will render a helper grid with step size of .1
    fractals.ciubix.xyz/fractals/Mandelbrot?width=256&height=256&position_x=-1.5&position_y=-.05&zoom=10&max_iterations=2000&colors=e50000,ff8d00,ffee00,028121,004cff,770088&num_colors=150&msaa=10&smooth=true&debug=true:

There's currently no caching so please be mindful that every time you do a request the gpu has to render a new image
